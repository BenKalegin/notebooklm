# IDM Sample Documents (Markdown)
This dataset contains synthetic sample documents for training / testing ERP-focused RAG over IDM-like content.
Folders represent practical IDM document categories.

## Categories
- **01_Logistics_Shipping**: Shipment/Receiving/Labels/Returns
- **02_Procurement_Sales**: PO/SO/RFQ/Quotes/Contracts
- **03_Finance_Accounting**: Invoices/Statements/Payments/Journals
- **04_Manufacturing_Planning**: Work Orders/MRP/Schedules
- **05_Asset_Service**: Maintenance/Service/Asset Records
- **06_Projects_PublicSector**: Project & Regulatory Records
- **07_HR_Workforce**: Timesheets/Schedules/Payroll
- **08_Quality_Compliance**: Inspections/Tests/CAPA
